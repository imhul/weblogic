**What's inside?**

-   **Frontend:**
    -   `parcel`
    -   `react` v18.2.0
    -   `redux`
    -   `helmet`
    -   `redux-thunk`
    -   `antd`
    -   `SCSS`
-   **Backend:**
    -   `netlify`
    -   `@netlify/functions`
    -   `undici`
    -   `reCapcha` v2
    -   `telegram bot`
